# PriceSentry デザイン仕様（UI/UX & ブランド）

## 1. ブランド
- **名称**：PriceSentry（外向け表記 “for eBay sellers”）
- **トーン**：信頼・迅速・賢い自動化
- **ロゴ方向性**：盾＋目＋折れ線（監視×保護×価格）、過度な値下げ印象は避ける

### 1.1 カラー（提案）
- Primary：#0A4DFF（信頼のブルー）
- Accent：#12B886（健全/成功）
- Gray：#3A3A3A / 背景 #F7F9FC
- 状態色：
  - 成功：#12B886
  - 警告：#F59F00
  - エラー：#E03131
  - 情報：#228BE6

### 1.2 タイポグラフィ
- 見出し：Inter/SF Pro（Bold）
- 本文：Inter/SF Pro（Regular/Medium）
- 数値：Tabular Lining（等幅）

## 2. 主要画面
### 2.1 ダッシュボード
- KPIカード：
  - **最安維持率**（%）
  - **当日改定数 / 失敗数**
  - **API消費（Browse/Trading/Inventory）**
- グラフ：日別改定数、最安維持率の推移
- アラート：上限接近、改定失敗、トークン期限

### 2.2 在庫一覧（Listings）
- 列：画像 / タイトル / 価格 / 状態 / ルール / 最安との差 / 最終改定 / 有効
- **最終改定の表示形式**: 相対時間（「2時間前」「昨日」「3日前」等）
- 行アクション：詳細、ルール編集、改定履歴
- フィルタ：カテゴリ、状態（New/Used/Excellent/Good）、価格調整（実行/停止）

#### 2.2.1 価格表示の仕様
- **全ての価格は送料込み**（「送料込み」の表記は不要）
- 表示形式：`$1,450 / 下限: $1,350`（改行）`ライバル: $1,449`
- ライバル価格 = 自分を除いた日本セラーの中で最も安い価格（価格＋送料の合計）
- **ライバル価格の文字色**：
  - 自分が最安の場合（差額がマイナス）→ 緑色
  - 自分が最安でない場合（差額がプラス）→ 赤色

#### 2.2.2 戦略（Strategy）列
- **役割**：その商品の価格を決める際に、どの競合を監視しているかを表示
- **表示タイプ**：
  - `検索: 日本セラー` - 日本セラー全体を検索して最安を監視
  - `商品ID: 12345...` - 特定の競合商品（Item ID指定）を監視

#### 2.2.3 価格調整トグル
- **役割**：その商品の自動価格調整機能のON/OFF
- **状態**：
  - ON（実行）→ 自動価格調整が有効、ライバルに合わせて価格を自動更新
  - OFF（停止）→ 自動価格調整が無効、価格は変更しない

### 2.3 ルール編集
- **タイプ選択**：
  - 競合ItemID監視（複数ID可）
  - 検索条件（キーワード、カテゴリ、フィルタ、JP限定）
- **セーフティ**：
  - 下限価格 / 最低利益（%/金額）
  - 改定幅（±% / 絶対値）
  - 1日改定回数上限 / 時間帯制限
- **ε（アンダーカット幅）**：1円/1¢単位
- **テスト実行**：即時チェック、結果プレビュー

### 2.4 改定履歴（Logs）
- 列：時刻 / 旧価格→新価格 / 比較根拠（競合価格合計・セラーID）/ 結果
- CSVエクスポート、API消費トレースへのリンク

### 2.5 設定
- トークン状態（更新期限）/ Marketplace-ID
- 巡回頻度（A/B/C配分、バックオフ許容）
- 通知（Slack, Email）
- チーム（メンバー招待、権限）

## 3. コンポーネント（shadcn/ui準拠）
- Card, Table, Badge, Tabs, Dialog/Drawer, Toast, Dropdown, Select, Input, Slider, Switch
- Chart：Recharts（折れ線・棒・面）
- Empty State：アイコン＋短文＋「設定に進む」CTA

## 4. UXルール
- 1画面＝1意思決定（編集はモーダル/ドロワー）
- 危険操作は2段階確認（改定幅超過・下限割れ）
- 状態の可視化：ルールのON/OFF、A/B/Cバッジ、トークン期限
- レスポンシブ：モバイルは主要KPIとルールON/OFFに特化

## 5. アイコン / 画像書き出し
- アプリアイコン：1024px正方（SVGベース）
- OG画像：1200×630（タイトル/サブコピー）
- ファビコン：32/64px

## 6. コピー（LP/アプリ内）
- ヒーロー：「日本セラーだけ見て、**常に最安へ。**」
- サブ：「**価格＋送料**で本当の最安を自動維持。最低利益は守ったまま。」
- CTA：「今すぐ在庫を接続」

## 7. アクセシビリティ
- コントラスト比 AA以上
- キーボード操作対応、フォーカスリング
- Live regionでトースト/進捗を告知
